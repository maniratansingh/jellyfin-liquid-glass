# Jellyfin Liquid Glass Theme

> **Personal Note:** This is a personal project created strictly for my own personal use. I created this because the default Jellyfin UI was just too basic for my liking and I needed something more premium and refined. If anyone else wants to use it, you are welcome to, but please note I built this solely for my own purposes.

> **⚠️ WARNING:** This is highly experimental. Things might break, layout glitches may happen, and updates to Jellyfin could break this theme. Use with caution.

## Credits & Inspiration
I really liked the design of the **Abyss Jellyfin Theme**, and I want to give full credit to the original creator, **AumGupta**. I used their work and built this entirely upon their code. This project is essentially a heavily modified version of their theme. You can find their original project here: https://github.com/AumGupta/abyss-jellyfin

## What to Expect (Compared to Default Jellyfin)
If you are coming from the standard, stock Jellyfin theme, here is what this theme changes:

- **Heavy Frosted Glass:** Replaces the flat solid colors with deep, translucent frosted glass (blur effects) across navigation bars, menus, and sidebars.
- **Apple TV-Style Video Player:** Completely overhauls the video player. Removes unnecessary clutter and adds a beautiful, centered glass "Orb" play button for a clean, cinematic experience.
- **Floating UI:** Sidebars and headers float above the background rather than being rigidly attached to the edges of the screen.
- **Refined Mobile Experience:** Shrinks down massive text and clunky icons on mobile devices so full movie names fit perfectly, and adds buttery-smooth animations when rotating your phone.

## How to use
Copy the link to the compiled CSS file and paste it into your **Jellyfin Dashboard -> General -> Custom CSS code** box:

```css
@import url('https://cdn.jsdelivr.net/gh/maniratansingh/jellyfin-liquid-glass@main/liquid-glass-bundle.css');
```
